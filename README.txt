less-markup
===========

LessMarkup Engine

* You have a serious team of developers, but don't have an experience in web development?
* You have brilliant web developers, but you know how hard it is to build quality web solution with serious functionality?
* You don't want to spend big resources to develop and support web project?
* You have already working .NET solution and you want to add web frontend?
* You are tired of fighting with html markup which is hard to test and lives its own life?

Then this solution is for you!

All that you need is to create your home page design, and we already did the rest for you. You can continue to concentrate on your business logic, you don't need to think on how link any html field to the field in the database. You can forget about fighting with html and css.
